Project Case Study

Newsly — AI-Powered Daily News Agent

An automated intelligence pipeline that aggregates, filters, and summarizes daily news across five custom categories, delivering optimized newsletters via SMTP and web dashboards.

Live Python • FastAPI • Groq Llama 3.3 • GitHub Actions
GitHub Live Dashboard
5 categories Tech, Finance, Politics, Sports, Fashion
Llama 3.3 70B model summarization
100% fully automated cron cycles

Overview

Newsly is an autonomous agent designed to combat news fatigue. It fetches the latest headlines from RSS feeds, deduplicates stories, feeds raw text to a high-speed Llama 3.3-70B model on Groq Cloud to synthesize concise summaries, and distributes clean, responsive newsletters. It also serves a modern FastAPI frontend for browsing past digests.

Need / Motivation

Staying informed involves navigating scattered websites, clickbait headings, and repetitive articles. By building an agentic filter, Mansi created a private, zero-maintenance news delivery workflow that extracts genuine signal from daily noise in seconds.

Objectives

  • Fetch global news stories across 5 core categories daily.
  • Execute fast, accurate language summarization with structured outputs using LLMs.
  • Deliver custom, responsive email newsletters automatically to subscribers.
  • Expose clean rest endpoints and hosting via FastAPI.

System Architecture

Newsly's serverless pipeline. GitHub Actions fires the daily trigger -> FastAPI fetches RSS -> Summarizes via Groq Llama 3.3 -> Dispatches to email lists and saves to dashboard cache.

Key Implementation Details

  • Prompt Engineering: Structured prompts forcing Llama 3.3 to output strict, consistent JSON arrays with headline titles, key bullet points, and source links.
  • FastAPI Deduplication: Employs headline token hashing to skip duplicate reports covering identical topics across multiple publishers.
  • SMTP Dispatcher: Delivers styling-rich HTML newsletters optimized for responsive display across various desktop and mobile email clients.

API Docs / Routes

GET /

Renders the responsive HTML homepage containing the most recently cached news categories and summaries.

POST /api/trigger

Private endpoint triggered by GitHub Actions CRON workflow to execute news harvesting, LLM calls, and newsletter dispatching.

Results

The system is deployed on Render and runs entirely on a free-tier setup. Summarizations are completed in under 2 seconds per category using Groq's high-speed endpoint. Digests are sent to subscribers every morning without fail, achieving 100% automated uptime.
